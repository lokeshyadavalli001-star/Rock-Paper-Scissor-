# Rock-Paper-Scissor-
A classic Rock Paper Scissors game implemented in Python. Play against the computer in this simple yet entertaining command-line game.
Overview

This is a simple implementation of the popular hand game. The user selects one of rock, paper or scissors, the computer makes a random choice, and the outcome is determined according to rules:
- **Rock** crushes **Scissors**
- **Scissors** cuts **Paper** -
 **Paper** covers **Rock**
